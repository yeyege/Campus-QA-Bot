## 8. 常见问题

### Q: 知识库重建后回答还是旧内容？

**原因：** 旧服务进程内存里仍是旧数据，未重启。

**解决：** 停止服务 → 重启 → 再问。

### Q: 图片加载不出来？

- 检查图片 URL 是否可达（浏览器直接打开测试）
- 桂林理工大学官网的外部图片依赖网络连通性
- 如果学校服务器加了防盗链，考虑下载到本地 `static/` 目录

### Q: API 返回 500？

- 查看终端日志查找具体报错
- 常见原因：API Key 过期、网络不通、模型接口异常
- 检查 `config/.env` 中的 `XIAOMI_API_KEY`

## 附录：准确率评估方法

### A.1 评估指标

| 指标 | 含义 | 计算方式 |
|------|------|----------|
| **检索召回率（Recall@K）** | 前 K 条检索结果中包含正确答案的比例 | `正确答案在top-K中的次数 ÷ 总测试题数` |
| **检索精度（Precision@K）** | 前 K 条检索结果中相关文档的比例 | `前K条中相关文档数 ÷ K` |
| **MRR（Mean Reciprocal Rank）** | 正确答案在检索结果中的平均排名倒数 | `1/排名 的均值，排名从1开始` |
| **答案准确率（Answer Accuracy）** | LLM 最终给出的答案是否正确 | `人工判断正确的次数 ÷ 总测试题数` |
| **幻觉率（Hallucination Rate）** | 知识库有答案但模型编造了错误信息的比例 | `编造次数 ÷ 有标准答案的题数` |
| **拒绝准确率（Rejection Accuracy）** | 对知识库外的问题正确拒绝回答的比例 | `正确拒绝次数 ÷ 无关问题数` |

### A.2 准备测试集

在 `docs/` 目录下创建 `test_set.json`，格式如下：

```json
[
  {
    "id": 1,
    "category": "图书馆",
    "question": "图书馆几点开门？",
    "expected_answer": "7:00-22:00",
    "keywords": ["7:00", "22:00"],
    "type": "factual"
  },
  {
    "id": 2,
    "category": "常用电话",
    "question": "保卫处电话是多少？",
    "expected_answer": "0773-5896110",
    "keywords": ["5896110"],
    "type": "factual"
  },
  {
    "id": 3,
    "category": "无关问题",
    "question": "今天天气怎么样？",
    "expected_answer": "拒绝回答",
    "keywords": [],
    "type": "out_of_scope"
  }
]
```

**字段说明：**
- `category`：所属 FAQ 分类
- `question`：测试问题
- `expected_answer`：预期答案关键内容（用于人工/自动判断）
- `keywords`：答案中必须包含的关键词
- `type`：问题类型，分为 `factual`（事实型）、`out_of_scope`（超出范围）、`multi_turn`（多轮）

**建议规模：**
- 每个 FAQ 分类至少 5 条（共 35+ 条）
- 无关问题 5~10 条
- 多轮对话场景 5 组

### A.3 自动化评估脚本

在项目根目录创建 `scripts/evaluate.py`：

```python
"""
准确率评估脚本
用法: python scripts/evaluate.py
"""
import json
import sys
import os
sys.path.insert(0, os.path.dirname(os.path.dirname(os.path.abspath(__file__))))

from src.core.agent import CampusAgent

# 加载测试集
with open("docs/test_set.json", "r", encoding="utf-8") as f:
    test_cases = json.load(f)

agent = CampusAgent()
results = []

for case in test_cases:
    q = case["question"]
    expected = case["expected_answer"]
    keywords = case.get("keywords", [])
    
    # 调用 Agent
    response = agent.chat(q, session_id="eval_session")
    answer = response["text"]
    
    # 关键词匹配判断
    keyword_hit = all(kw in answer for kw in keywords)
    
    results.append({
        "id": case["id"],
        "question": q,
        "answer": answer,
        "keyword_hit": keyword_hit,
        "type": case["type"]
    })
    print(f"[{'✓' if keyword_hit else '✗'}] {q[:30]}...")

# 统计
total = len(results)
factual = [r for r in results if r["type"] == "factual"]
oos = [r for r in results if r["type"] == "out_of_scope"]

print(f"\n=== 评估结果 ===")
print(f"总题数: {total}")
print(f"关键词命中率: {sum(r['keyword_hit'] for r in results)}/{total}")
if factual:
    acc = sum(r['keyword_hit'] for r in factual) / len(factual) * 100
    print(f"事实题准确率: {acc:.1f}%")
if oos:
    reject = sum(r['keyword_hit'] for r in oos) / len(oos) * 100
    print(f"拒绝准确率: {reject:.1f}%")
```

### A.5 人工评估维度

自动化只能测关键词命中，更准确的评估需要人工打分。评估维度如下：

| 维度 | 评分（1-5） | 说明 |
|------|------------|------|
| **准确性** | 1~5 | 答案是否完全正确，无事实错误 |
| **完整性** | 1~5 | 是否覆盖了问题的所有方面，无遗漏 |
| **相关性** | 1~5 | 答案是否紧扣问题，不跑题 |
| **友好度** | 1~5 | 语气是否亲切自然，像学长学姐 |
| **简洁性** | 1~5 | 是否简明扼要，不啰嗦 |

### A.7 MRR（平均倒数排名）计算

```bash
cd D:\agentproject
python -c "
import sys; sys.path.insert(0, '.')
from src.core.knowledge_loader import KnowledgeLoader

loader = KnowledgeLoader()
loader.init_chromadb()
loader.load_embedding_model()

test_set = [
    ('图书馆几点开门', '图书馆'),
    ('保卫处电话', '常用电话'),
    ('食堂支付方式', '校园生活'),
    ('转专业条件', '学院专业'),
    ('校医院位置', '设施服务'),
    ('学校有几个校区', '桂林理工大学'),
    ('选课时间', '教务管理'),
]

total_rank = 0
valid = 0
for query, expected_cat in test_set:
    results = loader.search(query, top_k=5)
    cats = [r['metadata']['category'] for r in results]
    if expected_cat in cats:
        rank = cats.index(expected_cat) + 1
        total_rank += 1.0 / rank
        valid += 1

mrr = total_rank / len(test_set)
hit_rate = valid / len(test_set) * 100
print(f'MRR (平均倒数排名): {mrr:.4f}')
print(f'Recall@5 (召回率): {hit_rate:.1f}%')
"
```

### A.8 评估标准参考

| 指标 | 优秀 | 良好 | 及格 | 不及格 |
|------|------|------|------|--------|
| 答案准确率 | ≥ 95% | ≥ 85% | ≥ 70% | < 70% |
| 检索召回率（Recall@5） | ≥ 95% | ≥ 85% | ≥ 70% | < 70% |
| MRR | ≥ 0.95 | ≥ 0.85 | ≥ 0.70 | < 0.70 |
| 幻觉率 | ≤ 2% | ≤ 5% | ≤ 10% | > 10% |
| 拒绝准确率 | ≥ 90% | ≥ 80% | ≥ 70% | < 70% |
