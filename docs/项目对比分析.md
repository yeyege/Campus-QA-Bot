## 二、核心功能缺失清单

### 高优先级（必须做）

| 编号 | 功能 | 描述 | 影响 |
|------|------|------|------|
| P1 | 多轮对话 | 记录对话历史，支持上下文理解 | 用户体验差 |
| P2 | 会话持久化 | 对话记录保存到文件/数据库 | 刷新丢失 |
| P3 | 用户反馈 | 点赞/点踩收集回答质量 | 无法优化 |
| P4 | 错误处理 | API失败时友好提示 | 用户困惑 |

### 中优先级（应该做）

| 编号 | 功能 | 描述 | 影响 |
|------|------|------|------|
| M1 | 知识库管理 | Web界面管理FAQ | 维护困难 |
| M2 | 意图识别 | 分类问题类型 | 回答不精准 |
| M3 | 测试用例 | 单元测试+集成测试 | 质量无保障 |
| M4 | 日志系统 | 记录请求和错误 | 排查困难 |

### 低优先级（可以做）

| 编号 | 功能 | 描述 | 影响 |
|------|------|------|------|
| L1 | Docker部署 | 容器化一键部署 | 部署麻烦 |
| L2 | 权限管理 | 用户登录注册 | 多用户场景 |
| L3 | 数据分析 | 问题统计、热点分析 | 优化无依据 |
| L4 | 知识库自动更新 | 分析未回答问题 | 知识库过时 |
